macFUSE 软件源自开源的 [Google MacFUSE](https://code.google.com/archive/p/macfuse/)，后者自 2009 年起已停止维护。

2011 年，Google MacFUSE 代码库被 fork，后续开发以 「FUSE for OS X」、后又以 「FUSE for macOS」的名义在 [osxfuse](https://github.com/osxfuse) GitHub 组织下继续。Apple 开始使用 OS X 作为营销名称之后，选用 **osxfuse** 项目名称，以区别于已放弃的 Google 项目；因原名与 Google 强关联，无法在原名下延续项目。

2019 年 macFUSE 的许可条款发生变更：将 macFUSE 与商业软件捆绑销售需取得商业授权。macFUSE 软件的若干组件不再开源。

在 Google MacFUSE 项目终止十余年后，加之许可变更与 macOS 命名演变，「FUSE for macOS」于 2020 年更名为 **macFUSE**。2021 年创建 [macfuse](https://github.com/macfuse) GitHub 组织，用于托管后续开发。
